# План внедрения Koin (Dependency Injection)

Этот план описывает шаги по интеграции Koin в Kotlin Multiplatform проект для управления зависимостями. Все изменения будут производиться в отдельной Git-ветке с пошаговыми коммитами и финальным слиянием.

## [ВЫПОЛНЕНО] Подготовка: Ветка Git
**Задача:** Создать новую ветку для работы над фичей.
- Создание ветки `feature/koin-integration`.

## [ВЫПОЛНЕНО] Этап 1: Подключение зависимостей
**Задача:** Добавить нужные библиотеки Koin в проект.
- Добавление версий и библиотек в `gradle/libs.versions.toml` (библиотеки `koin-core`, `koin-compose`, `koin-compose-viewmodel`).
- Подключение библиотек к `commonMain` в `composeApp/build.gradle.kts`.
- Синхронизация проекта в Android Studio.
- **Ожидаем аппрув -> Выполняем `git commit`**

## [ВЫПОЛНЕНО] Этап 2: Создание Koin-модулей
**Задача:** Описать граф зависимостей.
- Создание пакета `di` и файла `AppModule.kt`.
- Объявление синглтонов (`single`) для `HttpClient` и `NetworkMonitor`.
- Объявление `viewModelOf` / `viewModel` для `ServerStatusViewModel` и `MainViewModel`.
- **Ожидаем аппрув -> Выполняем `git commit`**

## [ВЫПОЛНЕНО] Этап 3: Инициализация Koin в приложении
**Задача:** Запустить Koin при старте приложения.
- Создание функции `initKoin()` для настройки фреймворка.
- Добавление корневой обёртки `KoinApplication` (в новых версиях Compose Multiplatform) внутри функции `App()` в файле `App.kt`.
- **Ожидаем аппрув -> Выполняем `git commit`**

## [ВЫПОЛНЕНО] Этап 4: Инжект зависимостей в UI
**Задача:** Убрать ручное создание объектов из `App.kt` и экранов.
- Удаление глобальных `val httpClient = ...`, `val mainViewModel = ...` из файла `App.kt`.
- Обновление экранов `App.kt`, `MainScreen.kt` и компонента `ServerStatusIndicator.kt`: теперь они будут запрашивать `ViewModel` через функцию `koinViewModel()`.
- **Ожидаем аппрув -> Выполняем `git commit`**

## [ВЫПОЛНЕНО] Этап 5: Слияние в основную ветку (Merge)
**Задача:** Влить фичу в основную ветку без Fast-Forward.
- Переключение на основную ветку (например, `main` или `master`).
- Выполнение команды `git merge --no-ff feature/koin-integration`.
- **Готово!**
